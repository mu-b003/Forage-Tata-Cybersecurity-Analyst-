# Forage - Tata Cybersecurity Analyst Job Simulation

This repository contains my work completed during the Tata Consultancy Services (TCS) Cybersecurity Analyst Job Simulation on Forage.

## Overview

The simulation focused on Identity and Access Management (IAM) within a large enterprise environment. The tasks included assessing IAM readiness, evaluating IAM strategies, designing IAM solutions, and creating an implementation plan.
