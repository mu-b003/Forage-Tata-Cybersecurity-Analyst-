# Forage - Tata Cybersecurity Analyst Job Simulation

## Overview

This repository contains my work completed during the Tata Consultancy Services (TCS) Cybersecurity Analyst Job Simulation on Forage.

The simulation focused on Identity and Access Management (IAM), including readiness assessments, IAM solution design, and implementation planning within a large enterprise environment.

## Tasks Completed

### Task 1: IAM Readiness Assessment

* Reviewed organizational IAM requirements.
* Evaluated security and business objectives.
* Identified IAM focus areas.

### Task 2: IAM Strategy Evaluation

* Assessed user lifecycle management.
* Evaluated access control mechanisms.
* Reviewed governance and compliance requirements.

### Task 3: IAM Solution Design

* Designed automated Joiner-Mover-Leaver processes.
* Proposed Role-Based Access Control (RBAC).
* Recommended Multi-Factor Authentication (MFA) and Single Sign-On (SSO).

### Task 4: IAM Implementation Planning

* Developed an IAM deployment roadmap.
* Identified integration challenges.
* Proposed mitigation strategies for legacy systems, cloud services, and third-party applications.

## Key Skills Demonstrated

* Identity and Access Management (IAM)
* RBAC
* MFA
* SSO
* User Lifecycle Management
* Risk Assessment
* Security Governance
* Enterprise Security Architecture

## Lessons Learned

This simulation improved my understanding of how IAM solutions are assessed, designed, and implemented in large enterprise environments. It also highlighted the importance of aligning cybersecurity controls with business objectives and operational requirements.
