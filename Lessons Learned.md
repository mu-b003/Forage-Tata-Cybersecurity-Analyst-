# Lessons Learned

## What I Learned

* IAM is important for managing identities and access at scale.
* RBAC helps simplify access management.
* MFA improves authentication security.
* Automated provisioning reduces manual effort.
* IAM should support both security and business objectives.

## Challenges

* Integrating legacy systems.
* Managing access across cloud services.
* Maintaining accurate user data.

## Future Learning Goals

* Microsoft Entra ID (Azure AD)
* Identity Governance and Administration (IGA)
* Privileged Access Management (PAM)
* Enterprise IAM Architecture
