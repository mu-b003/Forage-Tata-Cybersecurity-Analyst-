# Task 1 - IAM Assessment

## Objective

Assess TechCorp's readiness for IAM implementation.

## Key Findings

* TechCorp operates in more than 100 countries.
* The company has over 150,000 employees.
* Managing identities manually is difficult at this scale.
* User lifecycle management needs improvement.
* Access control processes require strengthening.
* Cloud services and legacy systems increase complexity.

## Recommendations

* Automate user provisioning and de-provisioning.
* Implement Role-Based Access Control (RBAC).
* Strengthen authentication using MFA.
* Improve IAM governance and auditing.

## Outcome

TechCorp is ready for IAM improvements but requires better automation, governance, and access management processes.
